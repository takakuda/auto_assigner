# AutoAssigner
Almost remove easy before_action

## Usage

```ruby
load_params(params)
```

include ApplicationController or before_action

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'auto_assigner'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install auto_assigner
```
